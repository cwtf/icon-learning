# Asset Library

Use this folder for source assets and website-ready exports.

## Folder Map

- `source/brand/logo/` - original company logo files, wordmarks, SVGs, AI/PDF exports.
- `source/brand/icons/` - source icon files or brand marks.
- `source/brand/fonts/` - licensed brand font files, if any.
- `source/brand/colors/` - brand guides, palettes, or reference screenshots.
- `source/photos/workshops/` - training, classroom, and workshop photography.
- `source/photos/team/` - staff and trainer photos.
- `source/photos/office/` - office, venue, or location photos.
- `source/photos/testimonials/` - approved customer/headshot photos.
- `source/logos/clients/` - original client logo files.
- `source/logos/partners/` - partner, certification, or authority logos.
- `source/programs/category-heroes/` - representative images for program categories.
- `source/programs/course-heroes/` - representative images for individual courses.
- `source/programs/slide-previews/` - screenshots or exports from course decks.
- `source/documents/certifications/` - certificates, accreditation proof, and related documents.
- `source/documents/hrd-corp/` - HRD Corp registration or claimable-training references.
- `web/` - optimized assets ready for the website build.

## Naming

Use lowercase kebab-case names:

```text
icon-learning-logo-primary.svg
hrd-claimable-training-workshop.webp
iso-9001-training-category-hero.webp
client-logo-company-name.svg
```

Prefer SVG for logos and WebP/AVIF for website photos. Keep original editable files in `source/`; place compressed website copies in `web/`.
